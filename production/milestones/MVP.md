# Milestone: MVP

**Target**: One playable life from birth to death with choices, branching tree replay
**Timeline**: 6 months (~8 sprints at 3 weeks each)
**Start Date**: 2026-03-23
**Target Date**: 2026-09-21

## Success Criteria

- [ ] Player can start a new life with random starting conditions
- [ ] Timeline scrolls through 8 life stages with variable pacing
- [ ] Minor events appear on the timeline (40+ templates, 600+ variations)
- [ ] Major events pause the timeline with Scenario Cards (20+ events)
- [ ] Choices modify character state with visible consequences
- [ ] End-of-life epitaph summarizes the life lived
- [ ] Basic branching tree shows paths taken and not taken
- [ ] Player can replay from any branch point in the tree
- [ ] All state persists in browser (IndexedDB)
- [ ] Web app runs as a PWA

## Sprint Breakdown (Planned)

| Sprint | Focus | Systems |
|--------|-------|---------|
| 1 | Foundation | Project setup, Character State Model, Life Stage Definitions, Event Content Database schema |
| 2 | Core Engine | App Shell, Timeline Engine, Save/State Persistence |
| 3 | Gameplay Logic | Starting Conditions Generator, Choice & Consequence System |
| 4 | Content Engine | Life Event Generator, initial event content authoring |
| 5 | Presentation | Event Presentation Layer (all 5 formats), integration testing |
| 6 | Content & Polish | Major event authoring (20 events), minor templates (40), playtest |
| 7 | Tree & Replay | Branching Tree Visualizer, branch replay, epitaph generation |
| 8 | Integration & Ship | Full integration, bug fixing, PWA setup, MVP release |
